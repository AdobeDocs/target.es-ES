# Integración de indicadores en la aplicación {#integrate}

Esta sección proporciona directrices de integración para todos los clientes de marcas, organizados por tipo de aplicación.

## Elija la ruta de integración {#choose}

Seleccione la guía que coincida con el tipo de aplicación:

1. [Guía de inicio](startup-guide.md) — Empiece aquí para obtener una descripción general de alto nivel de todos los pasos de integración
1. [Aplicaciones móviles](mobile-applications.md): integración de SDK para aplicaciones móviles
1. [Aplicaciones web](web-applications.md): integración de SDK para propiedades web

## Tipos de aplicaciones de un vistazo {#overview}

| Tipo de aplicación | Integración recomendada |
| --- | --- |
| **Aplicación móvil** | AEP Mobile SDK: consulte las guías de [Android](../sdk-releases/android/android-extension-integration-guide.md) y [iOS](../sdk-releases/ios/ios-extension-integration-guide.md) |
| **aplicación web** | Web SDK: consulte [Guías de integración web](../sdk-releases/web/web-extension-integration-guide.md) |
